# FND-005 — Add local developer orchestration

## Что сделано

- Добавлены root-команды локальной оркестрации:
  - `pnpm validate:env`
  - `pnpm healthcheck`
  - `pnpm up:dev`
- Добавлена env validation (`scripts/validate-env.cjs`):
  - проверка наличия `.env`
  - проверка обязательных ключей
  - проверка валидности и уникальности портов
- Добавлен baseline health-check (`scripts/healthcheck.cjs`) для `api/ui/bot/scheduler`.
- Обновлена root-команда `pnpm dev`:
  - теперь выполняет `validate:env` + `healthcheck` + параллельный запуск сервисов.
- Обновлен onboarding в `README.md` для запуска dev-среды одной командой.
- Задача отмечена выполненной в `BACKLOG.md`.

## Верификация

- `pnpm validate:env`
- `pnpm healthcheck`
- `pnpm quality`
- `pnpm up:dev`

Ожидаемый результат: окружение валидируется, baseline health-check проходит, сервисы стартуют одной командой.
