# FND-001 — Initialize multi-repo workspace structure

## Что сделано

- Создан workspace-каркас с сервисами `api`, `ui`, `bot`, `scheduler`.
- Для каждого сервиса создан единый baseline:
  - `README.md`
  - `LICENSE`
  - `.editorconfig`
  - `.env.example`
  - `package.json` (`build`, `test`, `dev`)
  - `src/index.js`
- Добавлены root-файлы workspace:
  - `package.json`
  - `pnpm-workspace.yaml`
  - `.nvmrc`
  - `.editorconfig`
  - `.env.example`
  - `README.md`
  - `LICENSE`
- Задача отмечена выполненной в `BACKLOG.md`.
- Добавлена запись в `changelog.md`.

## Верификация

- `pnpm install`
- `pnpm build`
- `pnpm test`

Ожидаемый результат: команды проходят для всех 4 сервисов.
