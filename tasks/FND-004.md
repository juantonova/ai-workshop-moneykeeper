# FND-004 — Configure CI templates for all repos

## Что сделано

- Добавлен reusable workflow `reusable-ci.yml` для единых quality gates.
- Добавлен общий workflow `ci.yml`, который применяет reusable шаблон ко всем пакетам:
  - `api`
  - `ui`
  - `bot`
  - `scheduler`
  - `contracts`
- В reusable CI включены обязательные стадии:
  - install (`pnpm install --frozen-lockfile`)
  - lint
  - test
  - build
- Добавлены обязательные элементы DOD:
  - matrix по Node (`20`, `22`)
  - проверка lockfile drift (`git diff --exit-code pnpm-lock.yaml`)
  - upload отчетов и артефактов

## Верификация

- Локально: `pnpm quality` проходит.
- В CI: на PR/Push запускается `.github/workflows/ci.yml`; падение любого этапа приводит к красному статусу PR.
