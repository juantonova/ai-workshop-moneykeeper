В роли principal software architect, проанализируй предоставленные PRD.md, TECHSPEC.md и сформулируй план реализации проекта.
Разбей его на крупные этапы на основании раздела Backlog Seeds (Epics) в TECHSPEC

Каждый этап разбей на задачи в формате:

```
- [ ] [AREA]-[001] Task Title
Task Context
Task DOD
```


Каждая задача - фактически промпт для coding agent, содержащий весь необходимый контекст
В конце каждого этапа добавь задачи на приборку, удаление ненужного, удаление дублирования, анализ безопасности и приведение кода в порядок.

Все задачи запиши в оптимальном для выполнения порядке в BACKLOG.md
